# .dotfiles

Configs and dotfiles.

## Prerequisites

- `zsh`
- `cargo` installed ([rustup](https://rustup.rs))
- `coreutils` installed (`brew install coreutils`)

## Installation

```bash
git clone git@github.com:norskeld/.dotfiles.git && .dotfiles/install.sh
```
